Video Call App Test

Instructions:

  1. Open http://localhost:3000 in two different browser windows/tabs
  2. Enter different names for each window (e.g., "User1" and "User2")
  3. Click "Start" in both windows
  4. Allow camera and microphone permissions when prompted
  5. In one window, click the "Call" button next to the other user's name
  6. Accept the call in the other window

Common Issues:

  • Call button not working: Check browser console for errors
  • No users showing: Make sure both windows are connected
  • Camera not working: Check browser permissions
  • Connection issues: Check if server is running on port 3000

Server Status:

Server should be running on port 3000

Check terminal for server logs

Browser Console:

Open browser developer tools (F12) and check the Console tab for any error messages.

Look for messages like:

  • "Connected to server with ID: ..."
  • "User registration: ..."
  • "Received user list: ..."
  • "startCall called with userId: ..."
